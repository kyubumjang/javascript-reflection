# 고양이 사진첩 만들기

## 구현 요구사항

- [ ] 고양이 사진 API를 통해 사진과 폴더를 렌더링합니다.
- [ ] 폴더를 클릭하면 내부 폴더의 사진과 폴더를 보여줍니다.
  - [ ] 현재 경로가 어디인지도 렌더링합니다.
- [ ] 루트 경로가 아닌 경우, 파일 목록 맨 앞에 뒤로가기를 넣습니다.
- [ ] 사진을 클릭하면 고양이 사진을 모달창으로 보여줍니다.
  - [ ] esc를 누르거나 사진 밖을 클릭하면 모달을 닫습니다.
- [ ] API를 불러오는 중인 경우 로딩 중임을 알리는 처리를 합니다.
